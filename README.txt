A simple way to add Google address autocomplete functionality to any form in WordPress. Limit the search to one country or worldwide.

== Description ==

Simple Address Autocomplete plugin adds Google's address autocomplete to form fields in WordPress. The functionality is added to form fields by adding a class to the input. The search can be limited to one specific country or worldwide. The pluign also allows users to get their visitors browser location. For help visit our knowledge base.

Features:

*   All WP forms supported
*   Limit search to one country
*   Get user geolocation.
*   A detailed knowledge base that provides answers to all your questions.

== Frequently Asked Questions ==

= Can I limit the search to one country? =

Yes. You can limit the search to one country or worldwide.

= Does it work with any form? =

Yes, even with custom forms. The knowledge base provides details on how to get it working with most popular forms in WordPress.

= Is it single line address or multi-line address autocomplete? =

It is a single line address autocomplete. 

= Does it support WooCommerce? =

No, it doesn't out of the box. However, out of the box option will be available in future updates.

= Can I add multiple address fields? =

Yes, you can.
